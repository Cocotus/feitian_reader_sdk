Thanks for reaching out to us about the reader iOS SDK.

1. For iOS app that integrates Fetian reader iOS SDK:
To develop iOS app that works with bR301-C18 reader via Bluetooth, customers' dev team needs to integrate Feitian reader iOS SDK:
https://github.com/FeitianSmartcardReader/FEITIAN_MOBILE_READERS/tree/master/iOS_SDK_ForApplePrimitiveCCID/3.5.71

In this Github repo, we have the SDK libs and the "iReader" demo project, customer's team can refer to this open-source code of iReader project to integrate the iOS sdk.
The latest version is v3.5.71.

The iReader demo project can just be opened in XCode and built to run in an Apple device which have Bluetooth capacity, and with a Feitian Bluetooth reader like bR301-C18.
And let me know if customer has any problem during the SDK integration.
